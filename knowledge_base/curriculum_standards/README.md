# 课程标准知识库

本目录用于存放各学科课程标准文件。

## 目录结构

```
curriculum_standards/
├── 高中/
│   ├── 语文课程标准.md
│   ├── 数学课程标准.md
│   ├── 英语课程标准.md
│   └── ...
├── 初中/
│   └── ...
├── 小学/
│   └── ...
└── 大学/
    ├── 计算机课程标准.md
    ├── 物理课程标准.md
    └── ...
```

## 文件格式

课程标准文件应为Markdown格式，包含以下内容：

```markdown
# [课程名称]课程标准

## 一、课程性质与地位

## 二、课程目标

### （一）知识与技能

### （二）过程与方法

### （三）情感态度与价值观

## 三、课程内容

## 四、实施建议
```

## 使用说明

1. 将课程标准文件放入对应的分类目录
2. Agent会自动检索相关课程标准用于教案设计
3. 可以添加自定义的课程标准
