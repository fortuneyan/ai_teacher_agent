# 教学经验知识库

本目录用于存放优秀教案、教学案例和教学经验总结。

## 目录结构

```
teaching_experience/
├── 优秀教案/
│   ├── Python教案示例.md
│   ├── 数学教案示例.md
│   └── ...
├── 教学案例/
│   ├── 课堂互动案例.md
│   ├── 实践项目案例.md
│   └── ...
└── 经验总结/
    ├── 教学心得.md
    ├── 学生反馈.md
    └── ...
```

## 使用说明

1. 存放优秀教案示例供Agent参考
2. 添加教学案例用于丰富教学内容
3. 记录教学经验帮助Agent学习
