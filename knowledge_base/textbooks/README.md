# 教材内容知识库

本目录用于存放各学科教材内容文件。

## 目录结构

```
textbooks/
├── 计算机/
│   ├── Python程序设计.md
│   ├── 数据结构与算法.md
│   └── ...
├── 数学/
│   ├── 高等数学.md
│   ├── 线性代数.md
│   └── ...
├── 语言/
│   ├── 英语.md
│   └── ...
└── 其他/
```

## 文件格式

教材内容应为Markdown格式，便于Agent理解和处理。

## 使用说明

1. 将教材内容文件放入对应的分类目录
2. Agent会检索相关教材用于教学设计
3. 建议按章节组织内容
